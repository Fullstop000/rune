# SNAP Decision Framework (SNAP 决策框架)

**Category:** Execution (执行落地)

**Framework ID:** #57

**Source:** https://pmframe.works/framework-snap

---

## Overview

面向现代忙碌买家（或用户）的决策影响框架。SNAP = Simple（简洁）、iNvaluable（不可替代）、Aligned（与目标对齐）、Priority（在决策优先时刻出现）。

---

## Four Dimensions

| Dimension | Description |
|-----------|-------------|
| **Simple** | 减少用户的认知负担和决策摩擦 |
| **iNvaluable** | 让用户感受到产品的独特价值 |
| **Aligned** | 产品价值主张必须与用户当前正在追求的目标紧密相关 |
| **Priority** | 在用户最有可能说「Yes」的时刻出现 |

---

## Three Decision Moments

| Moment | Description | Key Factors |
|--------|-------------|-------------|
| **「让我进入吗？」** | 用户第一次接触产品 | Simple 和 iNvaluable |
| **「要改变吗？」** | 评估是否切换现有工作方式 | Aligned 最关键 |
| **「选择你吗？」** | 多选项中做最终选择 | Priority 发挥作用 |

---

## Case Study: Slack (2013-2019)

**Simple + iNvaluable:** 注册后30秒内开始对话，可搜索的完整消息历史，与GitHub、Jira等工具深度集成。

**Aligned + Priority:** 最初针对技术团队推广，定价按活跃用户计费，在公司快速扩张期切入，借助「邀请同事加入」自然扩散。

description: Make faster product or sales decisions using SNAP criteria

# SNAP Decision Framework

Evaluate decisions through four lenses — Simple, iNvaluable, Aligned, Priority — to cut through complexity and decide faster.

## Steps

1. **Simple** — is the option easy to understand and implement? Strip unnecessary complexity.
2. **iNvaluable** — does it deliver exceptional value that justifies attention and resources?
3. **Aligned** — does it align with the customer's or organization's core objectives and beliefs?
4. **Priority** — is it urgent and important enough to act on now versus other options?
5. Score each option across all four dimensions; advance only options that pass all four.

## Output Format

SNAP scorecard for each option with pass/fail per dimension, overall recommendation, and rationale for the top choice.
