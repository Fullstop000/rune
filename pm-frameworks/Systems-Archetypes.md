# Systems Archetypes (系统基模)

**Category:** Systems Thinking (系统思维)

## Overview

Eight common system archetypes reveal recurring problem structures to anticipate traps.

**Creator:** Peter Senge ("The Fifth Discipline", 1990)

---

## Five Core Archetypes

| Archetype | Description |
|-----------|-------------|
| **Fixes That Backfire** | Symptomatic fixes create side effects that worsen original problem |
| **Limits to Growth** | Growth generates constraints that limit further growth |
| **Shifting the Burden** | Symptomatic solutions replace fundamental solutions, causing atrophy |
| **Eroding Goals** | Lowering goals rather than raising performance when gaps exist |
| **Escalation** | Mutual destruction from defensive actions triggering stronger responses |

---

## Case Study: MySpace (2006-2009)

**Double Trap: Fixes That Backfire + Shifting the Burden**

- 2006: 100M+ MAU, $12B valuation
- **Trap 1:** User complaints → more ads → worse experience → more churn → need more ads (death spiral)
- **Trap 2:** Poor tech → should rebuild (fundamental) → risky → rely on marketing (symptomatic) → capability atrophies
- **Result:** $12B (2007) → $35M (2011) — 99.7% decline in 3 years

# Systems Archetypes (Senge)

Identify common structural patterns in complex systems to anticipate unintended consequences and find high-leverage interventions.

## Steps

1. Map the situation as a causal loop diagram — identify reinforcing (R) and balancing (B) feedback loops.
2. Match against known archetypes: **Fixes That Backfire**, **Limits to Growth**, **Shifting the Burden**, **Eroding Goals**, **Escalation**, **Tragedy of the Commons**, **Growth and Underinvestment**.
3. Identify the archetype's signature behavior (e.g., short-term fix worsening long-term problem).
4. Locate the **leverage point** — the structural change that breaks the dysfunctional pattern.
5. Design an intervention targeting the leverage point rather than the symptom.

## Output Format

Causal loop diagram description, identified archetype with pattern match rationale, leverage point analysis, and recommended structural intervention.
