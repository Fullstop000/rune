## Overview

Google's UX metrics: Happiness, Engagement, Adoption, Retention, Task success—turning experience into trackable metrics.

**Creator:** Google

---

## Five Metrics

| Metric | Description |
|--------|-------------|
| **Happiness** | User satisfaction and attitude |
| **Engagement** | Depth of user interaction |
| **Adoption** | New users starting to use product |
| **Retention** | Existing users returning |
| **Task Success** | Efficiency, effectiveness, error rate |

---

## Process

1. **Define goals** — What success looks like
2. **Choose metrics** — Relevant HEART metrics
3. **Set baselines** — Current performance
4. **Measure** — Collect data over time
5. **Improve** — Take action based on insights

# HEART Framework (Google)

Track UX quality over time by measuring five dimensions: Happiness, Engagement, Adoption, Retention, and Task Success.

## Steps

1. Define the **product area** and user base you want to measure.
2. For each HEART dimension, select **metrics** that map to your product goals:
   - **Happiness** — satisfaction surveys, NPS, app store ratings
   - **Engagement** — session frequency, depth, content creation
   - **Adoption** — new users completing key onboarding steps
   - **Retention** — users returning after 1/7/30 days
   - **Task Success** — completion rates, error rates, efficiency
3. Set **baselines** for each metric using historical data.
4. Establish **goals** for improvement over a defined period.
5. Track weekly or monthly and report changes.

## Output Format

HEART scorecard table with columns: Metric, Baseline, Goal, Current, Trend (up/down/flat). Include a summary of key findings and recommended actions based on which metrics need attention.
