# OGSM Strategic Planning (OGSM 战略规划)

**Category:** Execution (执行落地)

**Framework ID:** #56

**Source:** https://pmframe.works/framework-ogsm

---

## Overview

A one-page strategic planning tool that connects long-term vision with daily actions through four hierarchical levels. Compresses 3-year strategy onto a single A3 page.

**Creator:** Various (management consulting)

---

## Four Levels

| Level | Description |
|-------|-------------|
| **Objective** | Qualitative, inspiring, 3-5 year vision |
| **Goals** | Quantitative, annual, verifiable metrics |
| **Strategies** | Directional choices with explicit "nots" |
| **Measures** | Leading indicators for monthly tracking |

---

## Key Principle

Four levels must form a complete causal chain. Forces explicit choices through "We will do X, NOT Y" format.

---

## Case Study: P&G Pampers Japan (1980s)

OGSM revealed strategy-market disconnect; discovered parents bought from pharmacies not supermarkets; 18-month reversal achieved.

# OGSM Strategic Planning

Translate vision into execution through four cascading layers: Objectives, Goals, Strategies, and Measures.

## Steps

1. Define the **Objective** -- a qualitative, aspirational statement of what you want to achieve
2. Set **Goals** -- 2-4 quantitative targets that define success for the objective
3. Choose **Strategies** -- the key approaches or bets you will make to reach goals
4. Define **Measures** -- specific KPIs, milestones, and dashboards to track each strategy

## Output Format

**Objective:** [Aspirational statement]

| Goal | Strategy | Measures |
|------|----------|----------|
| [Quantitative target 1] | [Approach to achieve it] | [KPIs / milestones] |
| [Quantitative target 2] | [Approach to achieve it] | [KPIs / milestones] |
| [Quantitative target 3] | [Approach to achieve it] | [KPIs / milestones] |
