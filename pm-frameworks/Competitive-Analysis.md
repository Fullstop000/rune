# Competitive Analysis (竞品分析框架)

**Category:** Definition (问题定义)

**Framework ID:** #18

**Source:** https://pmframe.works/framework-competitive-analysis

---

## Overview

系统评估直接竞品、间接竞品与替代方案，从功能、体验、商业模式、市场定位等维度绘制竞争全景图，找到市场空白与自身差异化的依据。

---

## Five-Step Process

| Step | Description |
|------|-------------|
| 1 | Define competitive scope using "Three Circles Model" (2-4 hrs) |
| 2 | Build standardized evaluation matrix (1-2 days) |
| 3 | Deep-dive experience core competitors (2-5 days) |
| 4 | Draw competitive positioning map (half-day) |
| 5 | Extract differentiation opportunities (half-day) |

---

## Six Analysis Dimensions

| Dimension | Description |
|-----------|-------------|
| Target users & positioning | Who are they targeting? |
| Core feature coverage | How complete is their offering? |
| User experience quality | How smooth is their UX? |
| Business model | How do they make money? |
| Market size & share | How big is their presence? |
| Technical & data moats | What keeps them competitive? |

---

## Competitor Types

| Type | Description |
|------|-------------|
| **Direct competitors** | Same target, same solution |
| **Indirect competitors** | Same target, different solution |
| **Substitute competitors** | Different target, same solution |

```markdown
description: Systematically compare competitors on features, positioning, pricing, and strategy
---

# Competitive Analysis Framework

Conduct a structured evaluation of competitors to identify market gaps and inform positioning.

## Steps

1. Identify direct, indirect, and potential competitors.
2. Gather data on each: features, pricing, target audience, positioning, go-to-market, funding, and growth signals.
3. Build a comparison matrix scoring competitors across key dimensions.
4. Analyze patterns — where do competitors cluster? Where are gaps or white space?
5. Derive strategic implications: differentiation opportunities, competitive threats, and positioning recommendations.

## Output Format

Competitor comparison matrix, feature parity heatmap, positioning map (2-axis plot), gap analysis, and strategic recommendations.
```
