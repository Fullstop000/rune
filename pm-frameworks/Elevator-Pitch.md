# Elevator Pitch (电梯演讲模板)

**Category:** Ideation (创意生成)

**Framework ID:** #31

**Source:** https://pmframe.works/framework-elevator-pitch

---

## Overview

A structured template enabling clear product communication in under 30 seconds. Conveys target users, core pain points, solutions, and differentiated value for quick alignment and interest.

---

## Template Structure (6 Components)

| Component | Description |
|-----------|-------------|
| 1. 目标用户 | Define core user group specifically |
| 2. 核心痛点 | Describe user's specific predicament |
| 3. 产品名称与类别 | Anchor with understandable product category |
| 4. 核心价值 | State concrete outcomes, ideally quantifiable |
| 5. 差异化 | Compare to known alternatives, one key difference |
| 6. 行动号召 (optional) | Specific next step |

---

## Refinement Process

1. Write long version (300-500 words), then compress
2. Read aloud and time test (25-40 seconds ideal)
3. Test on someone unfamiliar with the product domain
4. Prepare multiple versions per audience type
5. Update quarterly

---

## Applicable Scenarios

- Investment/media
- Cross-team alignment
- Early user recruitment
- PRD background sections
- New employee onboarding
- Competitions/demo days

---

## Case Study: Slack (2013-2014)

Focused on communication experience over feature lists. Differentiated through searchable history context within unified channels rather than email's CC chaos.
