# Business Model Canvas (商业模式画布)

**Category:** Systems Thinking (系统思维)

**Framework ID:** #77

**Source:** https://pmframe.works/framework-business-model-canvas

---

## Overview

Nine-building-block visual tool showing complete business model on one page—from "who do we create value for" to "how do we profit"

**Creator:** Alexander Osterwalder (Switzerland)

**Year:** 2004 (PhD thesis), 2010 (published)

---

## Nine Building Blocks

| Block | Description |
|-------|-------------|
| **Value Proposition (Center)** | What problem do you solve? What value do you create? |
| **Key Partners** | Who helps you do what you don't? |
| **Key Activities** | What must you do to deliver? |
| **Key Resources** | What assets do you need? |
| **Customer Relationships** | How do you interact? |
| **Channels** | How do you reach customers? |
| **Customer Segments** | Who do you serve? |
| **Cost Structure** | What are the biggest costs? |
| **Revenue Streams** | What do customers pay for? |

---

## Use Cases

1. Startup Initial Design - Sketch complete business in 30 minutes
2. Competitor Analysis - Compare strategic differences at a glance
3. Business Model Innovation - Change one block, observe ripple effects
4. Strategic Transformation - Systematically re-examine all assumptions
5. Investor/Board Presentations - One page vs. 50-slide decks
6. New Product Line Planning - Align cross-functional teams before execution

---

## Case Study: Nespresso

**Razor and Blades Model:**
- Machine (Razor): Low profit or loss—entry point
- Capsules (Blades): ~85% gross margin, $0.10 cost → $0.70-1.10 price
- Lock-in: Repeat purchase, 2-3 capsules daily per machine

---

## Five Usage Principles

1. Start with Customer Segments—all other blocks serve this
2. Format Value Proposition as "We help [who] solve [problem] through [solution]"
3. Cross-check Cost vs. Revenue—mark profit margins on each stream
4. Compare 3 Competitors side-by-side with color-coded differences
5. Color-code assumptions: Green = verified, Yellow = partially evidenced, Red = pure hypothesis

## Skill Download

**File:** `skills/business-model-canvas.md`

**Skill Content:**

```markdown
# Business Model Canvas (Osterwalder)

Visualize an entire business model on a single page using nine interdependent blocks.

## Steps

1. Define **Customer Segments** — who are you creating value for?
2. Articulate the **Value Proposition** for each segment.
3. Map **Channels** (awareness, purchase, delivery) and **Customer Relationships** (self-service, dedicated, community).
4. Identify **Revenue Streams** (per segment) and **Key Resources/Activities/Partners** needed to deliver.
5. Estimate **Cost Structure** and validate that revenues exceed costs sustainably.

## Output Format

A completed 9-block canvas table with concise entries per block, plus a brief viability summary noting the riskiest assumptions.
```
