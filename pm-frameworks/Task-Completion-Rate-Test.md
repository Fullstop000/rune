## Overview

Quantifies percentage of users successfully completing core tasks independently in given scenarios—most direct usability metric.

---

## Formula

**Task Completion Rate = (Users completing task / Total users) × 100%**

---

## Benchmarks

| Rate | Assessment |
|------|------------|
| **78%+** | Good |
| **90%+** | Excellent |
| **< 60%** | Needs significant improvement |

# Task Completion Rate Test

Quantify product usability by measuring what percentage of users can complete critical tasks. Use when you need a baseline usability metric or want to measure improvement after a redesign.

## Steps

1. **Identify key tasks** — select 3-5 tasks that represent core product value (e.g., "complete checkout," "create a report").
2. **Define success criteria** — specify exactly what counts as successful completion for each task.
3. **Recruit participants** — 5-20 users matching your target persona.
4. **Run the test** — observe each participant attempting each task; record success/failure and time on task.
5. **Calculate rates** — compute completion rate (%) and average time per task.
6. **Benchmark and act** — compare against industry benchmarks (78% average) and prioritize improvements for lowest-scoring tasks.

## Output Format

A results table with columns: Task, Completion Rate (%), Average Time, Failure Points, and Recommended Fix. Include before/after comparison if retesting.
