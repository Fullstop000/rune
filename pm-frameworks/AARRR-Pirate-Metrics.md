# AARRR Pirate Metrics (AARRR 海盗指标)

**Category:** Growth (增长策略)

**Framework ID:** #59

**Source:** https://pmframe.works/framework-aarrr

---

## Overview

Dave McClure 于 2007 年提出的增长漏斗模型，从获客到营收追踪五个关键环节，帮助团队识别最需优化的增长瓶颈。

**Creator:** Dave McClure

**Year:** 2007

---

## The Five Metrics

| Stage | Description | Key Question |
|-------|-------------|--------------|
| **A - Acquisition (获客)** | 用户从哪些渠道发现你？ | How do users find you? |
| **A - Activation (激活)** | 用户首次体验是否到达"啊哈时刻"？ | Do users have a great first experience? |
| **R - Retention (留存)** | 用户会回来吗？（最重要阶段） | Do users come back? |
| **R - Referral (推荐)** | 用户会主动分享产品吗？ | Do users tell others? |
| **R - Revenue (营收)** | 用户愿意付费了吗？ | How do you make money? |

---

## Methodology

五层漏斗每层回答一个核心问题并对应可测量指标。关键是找出哪层流失比例异常高。

---

## Usage Tips

- 先修留存再投获客，成本差10倍
- 每阶段只设一个北极星指标
- 找到"啊哈时刻"再优化激活
- 串行修复，逐一验证

# AARRR Pirate Metrics

Map and measure the user lifecycle across five funnel stages to find the biggest growth lever.

## Steps

1. **Acquisition** -- How do users find you? List channels and measure traffic/signups
2. **Activation** -- What is the "aha moment"? Define and measure first-value experience
3. **Retention** -- Do users come back? Measure D1/D7/D30 retention curves
4. **Revenue** -- How do you make money? Track conversion to paid and ARPU
5. **Referral** -- Do users invite others? Measure viral coefficient and referral rate
6. Identify the weakest stage and focus improvement efforts there
