# Reverse Brainstorming (逆向头脑风暴)

**Category:** Ideation (创意生成)

**Framework ID:** #25

**Source:** https://pmframe.works/framework-reverse-brainstorming

---

## Overview

Flip problem: first think "how to make product fail completely," diverge bad ideas, then flip each into valuable solutions.

---

## Process

1. **State the problem** — What you want to achieve
2. **Reverse the goal** — How to make it fail completely
3. **Generate bad ideas** — List ways to fail
4. **Flip each idea** — Convert failures into solutions
5. **Evaluate** — Assess the flipped ideas

---

## When to Use

- Creative problem-solving
- Risk identification
- Innovation workshops
- Finding hidden opportunities

# Reverse Brainstorming

Instead of solving the problem directly, ask "How could we make this problem worse?" then invert those ideas to find real solutions.

## Steps

1. **State the problem clearly** — Write the problem you want to solve in a single sentence.
2. **Reverse the problem** — Rephrase it as: "How could we cause this problem?" or "How could we make it worse?"
3. **Brainstorm reverse ideas** — Generate as many ways as possible to make the problem worse. Go wild; no filtering.
4. **Invert each idea** — Flip every "make it worse" idea into its opposite — a potential solution.
5. **Evaluate and select** — Review the inverted ideas, combine related ones, and select the most actionable solutions.

## Output Format

A three-column table: Reverse Idea (how to make it worse), Inverted Solution (the opposite), and Feasibility/Impact assessment. End with a prioritized shortlist of top solutions.
