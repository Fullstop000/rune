# North Star Metric (北极星指标框架)

**Category:** Validation (验证测试)

**Framework ID:** #37

**Source:** https://pmframe.works/framework-north-star-metric

---

## Overview

A single key metric that reflects the core value users derive from the product. Unifies team optimization direction and prevents departments from getting lost in local optimizations.

**Creator:** Sean Ellis, Growth Hacking Community

**Year:** ~2010

---

## Three-Layer Indicator System

| Layer | Description |
|-------|-------------|
| **Layer 01 - North Star Metric** | The sole metric defining success, directly reflecting the moment users gain core value |
| **Layer 02 - Input Metrics** | 3-5 leverage metrics driving the North Star, directly influenceable by each team |
| **Layer 03 - Output Metrics** | Revenue, growth, retention — natural outcomes after North Star improves |

---

## Key Standards

- Reflects the moment users gain core value
- Correlated with long-term business health
- Quantifiable and trackable
- Influenced by team actions
- **NOT** revenue (revenue is a result)
- **NOT** DAU (activity doesn't equal value)

---

## Company Examples

| Company | North Star Metric |
|---------|------------------|
| Airbnb | Nights Booked |
| Spotify | Monthly Music Listening Time |
| Facebook | DAU/MAU Ratio |
| Slack | Weekly Messages Sent |
| LinkedIn | Monthly Active Professionals |
| Duolingo | Daily Active Learners |

---

## Case Study: Spotify

**Problem:** Changed from MAU → Monthly Music Listening Time (2008-2012)
- MAU grew but users opened once and forgot

**Impact:** Drove podcast acquisition, birth of Discover Weekly, changed all team decision logic

## Skill Download

**File:** `skills/north-star-metric.md`

**Skill Content:**

```markdown
# North Star Metric

Define the one metric that reflects the core value your product delivers to customers. Use when your team lacks focus, tracks too many metrics, or needs alignment on what success looks like.

## Steps

1. **Articulate the value** — describe the moment users get value from your product.
2. **Brainstorm candidates** — list 5-10 metrics that could represent that value exchange.
3. **Filter** — a good North Star is measurable, reflects user value (not revenue), is a leading indicator, and the team can influence it.
4. **Select one** — choose the single best metric; define it precisely.
5. **Decompose** — break it into 3-5 input metrics that the team can act on directly.

## Output Format

A one-page brief with: the North Star metric name and precise definition, why it was chosen, its relationship to revenue, and a metric tree showing input metrics with current baselines and targets.
```
