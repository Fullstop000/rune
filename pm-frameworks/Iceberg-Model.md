# Iceberg Model (冰山模型)

**Category:** Definition (问题定义)

**Framework ID:** #19

**Source:** https://pmframe.works/framework-iceberg-model

---

## Overview

A systems thinking tool that reveals how visible events are merely surface symptoms of deeper structures. The model shows four levels: Events (above water), Patterns, Structures, and Mental Models (deepest underwater).

---

## Four Layers

| Layer | Description |
|-------|-------------|
| **Event Layer** | What is happening right now? Visible symptoms |
| **Pattern Layer** | Is this repeating? What trends exist over time? |
| **Structure Layer** | What systems, processes, or incentives create these patterns? |
| **Mental Model Layer** | What collective beliefs and assumptions create these structures? |

---

## Leverage Progression

Event layer = lowest leverage → Mental Model layer = highest leverage

---

## Case Study: Toyota Recall Crisis (2010)

- 8 million vehicles recalled, $4.8 billion in penalties
- **Root cause:** "speed over quality" mental model created supplier pressure structures
- Required CEO Akio Toyoda's personal intervention to reset core beliefs

## Skill Content

```markdown
# Iceberg Model

Use when looking beyond surface events to find systemic patterns and root structures.

## Steps

1. **Events (what happened?)** — Describe the visible event or symptom you are reacting to.
2. **Patterns (what trends recur?)** — Look for repeated occurrences over time. Is this a one-off or part of a trend?
3. **Structures (what causes the patterns?)** — Identify the systems, processes, incentives, or organizational structures that produce the patterns.
4. **Mental Models (what beliefs drive the structures?)** — Surface the assumptions, values, or beliefs that sustain the current structures.
5. **Choose your intervention level** — Decide where to act: reacting to events is fast but shallow; changing mental models is slow but transformative.

## Output Format

A four-layer diagram or table with rows for Events, Patterns, Structures, and Mental Models. Each row contains findings and evidence. Include a recommended intervention level with rationale.
```
