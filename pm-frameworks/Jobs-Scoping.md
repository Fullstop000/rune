# Jobs Scoping

**Category:** Definition (问题定义)

## Description

Starts from all possible user tasks to precisely scope product Job boundaries, preventing scope creep and direction drift.

## Process

1. **List all potential jobs** - Everything users might want to do
2. **Group related jobs** - Identify job categories
3. **Prioritize core jobs** - Focus on essential tasks
4. **Define boundaries** - Establish clear scope
5. **Validate** - Test against user needs

## When to Use

- Product scoping
- Feature prioritization
- Preventing scope creep
- Market analysis
